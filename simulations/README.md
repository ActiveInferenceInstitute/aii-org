# Interactive Simulations — Active Inference Institute

A collection of 12 interactive, browser-based standalone simulations demonstrating the core concepts of the Free Energy Principle and Active Inference.

## Catalogue

### Beginner
| File | Name | Key Concepts |
|------|------|--------------|
| `Ink.html` | Life Emerges – Living Ink | Free Energy, Self-Organisation, Markov Blanket |
| `Apple-Frog.html` | Apple or Frog? | Bayesian Inference, Prior Beliefs |
| `fep_surprise.html` | Surprise Minimisation | Prediction Error, Belief Updating |
| `school6.html` | Fish Schooling | Collective Agency, Generalised Synchrony |
