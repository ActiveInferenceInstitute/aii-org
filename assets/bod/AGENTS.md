# AGENTS.md — Board of Directors Headshots

## Purpose

Headshot photographs for the 10 Board of Directors members, displayed on `bod.html`.

## Conventions

- Format: JPEG (`.jpg`)
- Naming: `firstname_lastname.jpg` (lowercase, underscores)
- Referenced from `bod.html` as `src="images/bod/firstname_lastname.jpg"`
- Images are square or near-square, displayed at 88×88px (desktop) via CSS `object-fit: cover`
- All images include `loading="lazy"` and descriptive `alt` text
